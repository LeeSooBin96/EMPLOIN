요구사항4 - 검색 결과 보기
검색 결과
- 검색 건수 표시
- 검색 방법과 기준은 본인 방식
- 검색 알고리즘 사용 안해도 됨
- 검색 키워드 예)C++

기본 검색 이외에 상세 검색 기능의 조건은 최소 3개 이상
예) 지역, 경력, 학력, 회사 등

공고 검색 결과 리스트에 회사명, 공고제목과 간단한 직무 설명이나 추가정보 포함

요구사항5 - 공고 상세 보기
검색 결과 중 공고를 선택하면 해당 공고의 상세 내용 볼 수 있도록 구성
간단한 회사 소개 - 담당 업무 - 포지션 - 자격요건 - 근무조건 - 복지혜택 - 전형절차 - 유의사항 - 모집요강 - 근무시간 - 스킬 - 우대조건 등

요구사항6 - 기업 정보 보기
검색된 공고에서 기업 정보를 선택하면 기업 정보 볼 수 있도록
기업 소개 - 설립일,년차 - 기업형태 - 사원수 - 매출액 - 업종 - 홈페이지 - 주소 - 대표자명 - 사업내용 - 인원 증감추이 - 재무정보 등
